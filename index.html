Flappy Bird Clone

Play this JavaScript implementation of the PyGame code

Tap anywhere on the screen to make the bird flap!

How to Play

Press the SPACEBAR or tap the screen to make the bird flap and avoid the pipes.

Each pair of pipes you pass gives you 1 point.

If you hit a pipe or the ground, the game ends.
